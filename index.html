CSA Final Project: Tetris

A showcase of my Tetris clone I made for my Computer Science A class Junior year. I used the Processing library for the graphics as I wasn't as knowledgeable as I am now about java graphics. This is a very basic version of Tetris. I didn't implement advanced piece rotation or things like that, but I does play like normal Tetris and I was proud when I made this.
